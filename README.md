## 🎮 Crossy Road - Demo

A 3D endless runner inspired by Crossy Road, developed in Unity featuring:

- Procedurally generated terrain
- Dynamic obstacle spawning
- Moving vehicles with collision detection
- Character movement and jump animations
- Particle effects and stylized visual effects
- UI system with score tracking
- Win and Game Over screens
- 3D models, prefabs, and shaders
