# Unity Gameplays

A collection of beginner game development projects created with Unity, exploring both 2D and 3D gameplay mechanics, game systems, and level design.

## 🎮 Breakout Game - Demo

A classic 2D arcade-style game developed in Unity featuring:

- Paddle growth power-up
- Paddle shrink power-down
- Ball duplication power-up
- Collision and scoring systems
- Simple arcade gameplay mechanics

## 🎮 Crossy Road - Demo

A 3D endless runner inspired by Crossy Road, developed in Unity featuring:

- Procedurally generated terrain
- Dynamic obstacle spawning
- Moving vehicles with collision detection
- Character movement and jump animations
- Particle effects and stylized visual effects
- UI system with score tracking
- Win and Game Over screens
- 3D models, prefabs, and shaders
