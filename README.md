## 🎮 Breakout Game - Demo

A classic 2D arcade-style game developed in Unity featuring:

- Paddle growth power-up
- Paddle shrink power-down
- Ball duplication power-up
- Collision and scoring systems
- Simple arcade gameplay mechanics
